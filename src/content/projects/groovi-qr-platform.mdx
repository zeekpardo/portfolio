## Project Overview

**Groovi QR & NFC** is a SaaS platform that transforms static QR codes into dynamic, manageable business tools. Unlike traditional QR codes that point to one destination forever, Groovi enables real-time updates, scheduling, and campaign management.

**Live Platform:** [groovi.cc](https://groovi.cc)

## Key Features

- **Dynamic Playlists**: One QR code cycles through multiple destinations automatically
- **Smart Scheduling**: Time-based destination switching (menu → reservations → specials)
- **Campaign Switching**: Instantly change QR actions (website → phone → WhatsApp → email)
- **Multi-Tenant Architecture**: Separate workspaces for different organizations
- **Team Collaboration**: Multi-user support with role-based permissions and team management
- **Unified Management**: Control both QR codes and NFC tags from one dashboard
- **Real-time Analytics**: Track engagement across all touchpoints

## Tech Stack

### Frontend
- **Framework**: Next.js with React
- **Styling**: Tailwind CSS for responsive design
- **State Management**: Zustand for client-side state
- **UI Components**: Custom component library

### Backend
- **Runtime**: Node.js
- **API**: RESTful services with real-time updates
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: Clerk for user management and multi-tenant authentication
- **Payments**: Clerk for subscription billing and payment processing

### Infrastructure
- **Hosting**: Vercel for frontend deployment
- **Database**: Supabase for PostgreSQL hosting
- **CDN**: Global distribution for QR redirect speed
- **Analytics**: Custom tracking system for engagement metrics

### Additional Tools
- **Design**: Figma for UI/UX mockups
- **Project Management**: Linear for task tracking
- **Version Control**: Git with GitHub
- **Monitoring**: Custom dashboard for platform health
